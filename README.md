### Installation

Install [Python 3.6.2](https://python.org)

after install you can check your python version

```sh
$ python --version
```
or
```sh
$ python3 --version
```

#### Start program
For start release:
```python
from Yandex import Yandex
yandex = Yandex()
yandex.auth('Username', 'Password')
print(yandex.profile)
```
